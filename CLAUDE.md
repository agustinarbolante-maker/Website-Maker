# Website Maker - Claude Code Guidelines

## Git & GitHub Workflow

**IMPORTANT:** As work progresses on this project, you MUST:

1. **Commit regularly** - After completing meaningful features, bug fixes, or sections of work, create a local git commit with a clean, descriptive message
2. **Use clean commit messages** - Follow these patterns:
   - Feature: "Add [feature name]" (e.g., "Add navigation menu")
   - Fix: "Fix [issue]" (e.g., "Fix responsive layout on mobile")
   - Update: "Update [component/file]" (e.g., "Update button styling")
   - Keep messages concise and meaningful
3. **Push to GitHub regularly** - After committing, push changes to GitHub (https://github.com/agustinarbolante-maker/Website-Maker) to maintain a backup and history
4. **Never lose work** - Regular commits and pushes ensure we always have a saved version of the project
